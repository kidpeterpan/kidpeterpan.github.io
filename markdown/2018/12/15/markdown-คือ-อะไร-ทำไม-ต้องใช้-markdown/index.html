    Markdown คืออะไร ?





    markdown คือ markup language แบบหนึ่งที่ ที่ design มาเพื่อให้ง่ายต่อการเขียน content เราสามารถจัดการกับ format ต่างๆ ของ content รวมไปถึงรูปภาพและ link ได้โดยง่ายดาย markdown ถูกคิดค้นโดย John Gruber และ Aaron Swartz ในปี 2004 ซึ่งเราสามารถพบเห็น markdown ได้บ่อยครั้งกับ file readme ของ project ต่างๆ file หรือ website หลายๆ แห่งก็รองรับการเขียน content ด้วย markdown file ของ markdown นั้นจะลงท้ายด้วย .md หรือ .markdown สามารถดูตัวอย่าง syntex หรือลองเขียน markdown ได้ที่ https://dillinger.io/

    ทำไมต้องใช้ Markdown ?

    คำถามนี้ผมว่าก็ถือว่าตอบยากพอสมควร ผมจะพูดว่าทำไมผมถึงเลือกใช้ markdown แทนแล้วกัน

    1. Github

    github แทบจะเป็นที่แรกเลยที่ผมได้เห็นไฟล์ .md ใช่แล้วเพราะบนหน้าเว็บของ gihub เองก็ support markdown ทำให้คนที่เขียนโปรแกรม สามารถเขียนคู่มือของตัวเองได้สะดวกด้วย file README.md ส่วนคนที่เข้ามาก็สามารถอ่านคู่มือการใช้งานได้สะดวกเช่นกัน ดังนั้นการจะเขียนคู่มือที่สวยงาม แล้วเป็นระเบียบได้ developer ทุกคน ก็จำเป็นต้องเรียนรู้ syntex ของ markdown

    2. Static Web Generator

    ผมเองมีความสนใจด้านงานเขียนอยู่บ้าง เพื่อที่จะเป็นการทบทวนความรู็ของตัวเอง และส่งต่อความรู้สู่ผู้อื่นไปในเวลาเดียวกัน ผมสนใจ platform เช่น medium wordpress ถ้าใช้ platform อย่าง medium เราก็ไม่ต้องมี host เอง แต่กับ wordpress ไม่ใช่ แล้วปัญหาที่คนส่วนใหญ่เจอก็คือเมื่อมีคนใช้งานเยอะ มันก็จะช้า static web generator คืออะไร คือมันเหมือนเป็น framework แบบหนึ่งที่เราเขียนด้วย markdown แล้วมันจะ generate เป็น static html ให้ แถมเรายังนำมันไป deploy บน host ฟรีแบบ github page ได้อีกด้วย ทำให้เราไม่ต้องมี database หรือ process ที่ฝั่ง serverside เลย ทำให้เว็บตอบสนองได้เร็วมาก และ markdown ก็กลายมาเป็นอีกทางเลือกหนึ่งที่น่าสนใจ สำหรับคนที่อยากมี blog เล็กๆ เป็นของตัวเอง แล้วสามารถ custom หน้าเว็บได้ตามใจ ไม่ยึดติดกับ platform ใดๆ ตัวอย่างของ static web generator ก็เช่น hugo, middleman และ jekyll เป็นต้น

    3. Editor

    editor หลายตัวตอนนี้รองรับการเขียนด้วย markdown ไม่ว่าจะเป็น vscode, atom, sublime ซึ่งบางครั้งเราสามารถใช้ markdown เพื่อจด note ให้มีความสวยงามและเป็นระเบียบได้ แถมมันยังเบามากๆ อีกด้วย

    ตัวอย่าง

    Header
    เราสามารถสร้าง header ได้จากการเขียน

    # แล้วตามด้วยหัวข้อที่ต้องการเขียน

    โดยจะเป็น # หรือ ##.. มากกว่า 1 ตัวก็ได้ # 1 ตัวจะให้ค่าคือ html h1 # 2 ตัวจะได้ค่าเป็น h2 เป็นต้น

    # This is haader.

    ผลลัพธ์คือ

    This is header.

    Line Break
    การขึ้นบรรทัดใหม่ สามารถทำได้โดยเว้นวรรค 2 space แล้ว enter
    ตัวอย่างแรกคือ enter เลยโดยไม่เว้นวรรค 2 space

    ลองดูว่าจะขึ้นบรรทัดใหม่ไหม
บรรทัด1
บรรทัด2

    ผลลัพธ์คือ
    ลองดูว่าจะขึ้นบรรทัดใหม่ไหม บรรทัด1 บรรทัด2 ตัวอย่างที่ 2 คือ เว้นวรรค 2 space แล้ว enter

    ลองดูว่าจะขึ้นบรรทัดใหม่ไหม
บรรทัด1
บรรทัด2

    ผลลัพธิ์คือ
    ลองดูว่าจะขึ้นบรรทัดใหม่ไหม
    บรรทัด1
    บรรทัด2

    TODO List
    เราสามารถสร้าง todo list ได้โดย

    - [ ] ตัวอย่าง todo list แบบไม่ถูกติ๊ก
- [x] ตัวอย่าง todo list แบบถูกติ๊ก

    ผลลัพธ์คือ

    • ตัวอย่าง todo list แบบไม่ถูกติ๊ก
    • ตัวอย่าง todo list แบบถูกติ๊ก
